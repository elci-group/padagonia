# Security Policy

## Supported Versions

Only the latest published version of PADAGONIA receives security updates. We
recommend staying up to date with the main branch or the most recent release.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you discover a security issue in PADAGONIA, please report it privately to the
maintainers. Do not open a public issue or pull request for security
vulnerabilities.

Email: agent@elci.group

Please include:

- A description of the vulnerability.
- Steps to reproduce or a proof of concept.
- The affected version(s) or commit.
- Any suggested remediation.

We aim to acknowledge reports within 5 business days and will keep you informed
as we investigate and resolve the issue.
